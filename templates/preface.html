Préface

Avant d'entreprendre l'acquisition de compétences avec CSS, JavaScript, PHP et MySQL il est important d'avoir quelques connaissances de base sur le HTML. Le HTML est le langage qui permet d'identifier les différents éléments d'une page Web (titre, tableau, image, paragraphe, etc.). Je vous conseille donc de consulter le projet Héritage de Marc-Antoine Noreau-Marois : HTML 101. De plus, si vous n'avez vraiment aucune base sur la création d'un site Web, je vous conseille aussi de consulter le projet Hériage de Vincent Couture : Comment faire un site web pour les nuls.

Lors d'une première visite, il est conseillé de consulter la documentation dans l'ordre présenté dans le menu. De plus, il ne faut pas s'attendre à comprendre entièrement dès une première lecture rapide! C'est peut-être seulement une introduction au développement Web, mais c'est aussi une immersion complète dans un tout nouvel univers assez abstrait pour la plupart des gens n'ayant jamais programmé. Je crois aussi, par expérience personnelle, que la meilleure façon d'apprendre est de se pratiquer et d'essayer de réaliser des choses malgré notre connaissance parfois bien faible. Je recommande aussi de ne pas sauter des parties, car cela pourrait gravement altérer votre compréhension pour le reste du contenu. Chacune des sections contient une table des matières pour vous donner un avant-goût de son contenu et afin de s'y retrouver plus rapidement lors de consultations ultérieures.
